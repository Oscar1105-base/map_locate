## 前情提要

Python 3.11 環境

### 主要功能

這專案中包含以下功能：

- 基礎座標定位 + 設置收尋範圍
- 3個演算法( SA、ABC、自研複合演算法)
- 更新店家資訊透過視窗或csv & txt匯入+格式轉換
- 繪製地圖與收尋店家圖標，輸出成html檔案
